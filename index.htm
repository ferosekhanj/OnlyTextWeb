Only Text Editor

Welcome!!

This is a simple distraction free text editor that lets you concentrate only on your content. It runs inside your web browser. Your content is auto saved to your browser's local storage. You format your text easily and the editor takes care of converting it to html. See the formatting section below. Start typing your content in full screen with the basic formatting and once done just toggle the html preview. Copy and paste the html content to your target location (blog, html file, sharepoint etc.,).

 Heading

For heading precede the content with !

For eg

!This is level one heading
!!This is level two heading
!!!This is level three heading
!!!!This is level four heading
!!!!!This is level five heading
!!!!!!This is level six heading

 List

For list precede the content with a #

# This is a level one list
## This is a sub list under level one list
## This is a sub list under level one list
### This is a sub list under level two list
### This is a sub list under level two list
### This is a sub list under level two list
## This is a sub list under level one list
# This is level one list item again

 Misc

For horizontal ruler just add --- in a line

---

For preformatted content enclose the content between {{ }}

{{ Any content here will appear as is }}
