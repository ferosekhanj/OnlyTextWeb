Only Text Help

This is a simple text editor that lets you concentrate only on your content. It runs inside your web browser. Your content is auto saved to your browser's local storage. You format your text easily and the editor takes care of converting it to html. See the formatting section below. Start typing your content in full screen with the basic formatting and once done just toggle the html preview. Copy and paste the html content to your target location (blog, html file, sharepoint etc.,).

Click here to open only text editor

Shortcut Keys

Ctrl + b - Convert the content to formatted html.
Ctrl + F2- Increase the font size.
Ctrl + F3- Decrease the font size.
Ctrl + x - Cut.
Ctrl + c - Copy.
Ctrl + v - Paste.
Ctrl + z - Undo.
Ctrl + y - Redo.
